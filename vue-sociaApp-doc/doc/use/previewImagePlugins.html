previewImagePlugins

previewImagePlugins

图片全屏预览插件

引用组件

import {previewImagePlugins} from './plugins'
Vue.use(previewImagePlugins)

使用组件

this.$previewImage.show({
  index,
  urls
})

参数说明

参数 类型 示例 说明
index Number 默认：0 当前显示图片的索引
urls Array [] 需要预览的图片http链接列表
